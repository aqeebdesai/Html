Recomended folder structure

Root directories:

Main folder containing all website files

html files:

store main .html files at the root level for easy access

css folder:

create a css/ folder for all cascading style sheets

Js folder:

use a scrips/ folder for the javascript files

Images folder:

Store images in images/ folder

Assets folder:

Other assets heavy like fonts/videos can go in assets folder

sub-directories

for multiple page websites using folder to categorize content
